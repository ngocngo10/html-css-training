## Getting started
* Clone Repo: 
> git clone https://github.com/ngothingoc10/html-css-training.git
* Change the path to the folder you just cloned to your computer
> cd html-css-training
* Pull branch:
> git pull origin practice-two
> cd practice-two
* Install npm
> npm install
* Build Project:
> npm run build
* Run Project:
> npm start
